# Real-time Collaboration Tool

## Structure
- backend/: Node.js + Express + Socket.IO server
- frontend/: Vite + React client

## Run locally
1. Backend
   cd backend
   npm install
   npm run dev

2. Frontend
   cd frontend
   npm install
   npm run dev

Open the frontend URL (usually http://localhost:5173), open two browser windows and join same room id.
